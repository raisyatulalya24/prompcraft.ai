# PromptCraft AI

AI-powered prompt engineering platform built with PHP, MariaDB, JavaScript, HTML, and CSS.

## Features

- User Authentication
- AI Prompt Generator
- Prompt Improver
- Prompt Testing
- Analytics Dashboard
- Favorites & History
- Responsive Design

## Technologies

- PHP
- MariaDB
- HTML
- CSS
- JavaScript
- Groq API

## Setup

1. Clone this repository.
2. Configure your database.
3. Open `api.php`.
4. Replace:

```php
YOUR_GROQ_API_KEY
```

with your own Groq API Key.
